# 📊 Análise de Dados com Python - Regressão Linear

Este projeto faz parte dos meus estudos em Análise de Dados com Python.

##  Objetivo
Aplicar regressão linear para prever o preço de carros com base no consumo de combustível (highway-mpg).

##  Tecnologias utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

##  O que foi feito
- Importação e exploração dos dados
- Criação de modelo de regressão linear
- Treinamento do modelo
- Geração de previsões
- Análise de coeficientes

##  Aprendizados
Este projeto reforçou conceitos de machine learning, especialmente regressão linear e manipulação de dados.
